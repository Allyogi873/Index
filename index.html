Code Camp - 2020 Code Camp es un evento virtual organizado por la comunidad [Microsoft Dev Group Cobán](https://www.facebook.com/groups/MsDevGroupCoban), orientado a desarrolladores de software que buscan aprender nuevas habilidades de desarrollo. Nuestro objetivo es mejorar tus habilidades técnicas preparándote para el éxito con las herramientas adecuadas, ya sea para mejorar en tu trabajo actual, o prepararte para tu siguiente reto profesional. Gracias al apoyo de la comunidad [Guatemala SQL Server User Group](https://www.facebook.com/groups/gtssug/).
